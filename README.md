# Hye House Website

A modern, culture-forward website for Hye House, an Armenian-led creative community.
Built with Next.js, Tailwind CSS, and Framer Motion.
Inspired by Mouthwash Studio.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: Page routes (Home, Events, Merch, Story, Contact).
- `src/components`: Reusable UI components organized by feature.
- `src/data`: Mock data for events and merch.
- `src/app/globals.css`: Global styles and Tailwind v4 theme configuration.

## Design System

- **Colors**: Carbon Black, Optic White, Concrete Grey, Smoke.
- **Typography**: Inter (Google Fonts).
- **Vibe**: Dark mode, editorial, minimal, bold.
